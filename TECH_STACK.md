## 2. 技术栈选型

### 2.1 前端技术栈

| 技术组件 | 选型 | 版本 | 说明 |
|---------|------|------|------|
| **框架** | Vue 3 | 3.4+ | 渐进式JavaScript框架，组合式API |
| **构建工具** | Vite | 5.0+ | 快速构建工具，开发体验好 |
| **UI组件库** | Element Plus | 2.5+ | Vue 3组件库，开箱即用 |
| **状态管理** | Pinia | 2.1+ | Vue官方推荐状态管理库 |
| **路由** | Vue Router | 4.2+ | 官方路由管理器 |
| **HTTP客户端** | Axios | 1.6+ | HTTP请求库 |
| **PDF生成** | html2pdf.js | 0.10+ | HTML转PDF |
| **图片上传** | Element Plus Upload | 内置 | 支持拖拽、预览 |

**选型理由**:
- Vue 3生态成熟，开发效率高
- Element Plus组件丰富，降低开发成本
- Vite构建速度快，热更新即时
- html2pdf.js功能稳定，满足PDF导出需求

### 2.2 后端技术栈

| 技术组件 | 选型 | 版本 | 说明 |
|---------|------|------|------|
| **框架** | Flask | 3.0+ | 轻量级Web框架，适合快速开发 |
| **异步支持** | Flask-CORS | 4.0+ | 跨域请求支持 |
| **ORM** | SQLAlchemy | 2.0+ | Python SQL工具包 |
| **数据库** | MySQL | 8.0+ | 关系型数据库，稳定可靠 |
| **缓存** | Redis | 7.0+ | 内存数据库，提升性能 |
| **任务队列** | Celery | 5.3+ | 异步任务处理（可选） |

**选型理由**:
- Flask轻量灵活，API开发效率高
- MySQL成熟稳定，适合存储订单和用户数据
- Redis缓存热点数据，提升响应速度

## 5. 开发环境搭建

### 5.1 后端环境

```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 安装依赖
cd backend
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入API Key等信息

# 初始化数据库
python scripts/init_db.py

# 启动服务
python app/main.py
```

### 5.2 前端环境

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 7. 性能优化

### 7.1 缓存策略

- **Redis缓存**: 缓存热点故事模板
- **CDN加速**: 图片和PDF通过CDN分发
- **OSS预热**: 热点图片提前推送到CDN节点

### 7.2 异步处理

- **Celery任务队列**: 图片生成使用异步任务
- **WebSocket推送**: 实时推送生成进度

### 7.3 负载均衡

- **Nginx反向代理**: 多实例负载均衡
- **数据库读写分离**: 读写请求分离

## 8. 安全措施

### 8.1 内容安全

- **敏感词过滤**: 10万+违规词库
- **图片审核**: 调用阿里云内容安全API
- **人工抽检**: 10%订单人工审核

### 8.2 数据安全

- **HTTPS加密**: 全站HTTPS
- **数据库加密**: 敏感信息加密存储
- **访问控制**: API鉴权

### 8.3 隐私保护

- **照片自动删除**: 7天后删除原始照片
- **用户授权**: 明确授权使用条款
- **数据最小化**: 只收集必要信息
